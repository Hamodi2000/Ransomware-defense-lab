# Results Summary

## Detection Effectiveness

The baseline configuration using Microsoft Defender detected several ransomware samples but failed to detect others. Some samples were able to execute and encrypt the system.

In contrast, multilayered security configurations detected and blocked all ransomware samples tested during the experiment.

Network monitoring using Suricata also detected malicious traffic generated by ransomware samples attempting to communicate with external servers.

---

## Detection Speed

Detection times varied depending on the security solution used.

Average detection times observed:

* Windows Defender: ~1.21 seconds
* Avast: ~1.16 seconds
* Bitdefender: ~0.10 seconds
* ZoneAlarm: ~0.04 seconds
* Malwarebytes: ~0.028 seconds

These results indicate that layered security solutions can significantly improve response times.

---

## Deletion and Remediation

Removal times also varied across security tools.

Average deletion times:

* Windows Defender: ~40 seconds
* Avast: ~7.7 seconds
* Bitdefender: ~45 seconds
* Malwarebytes: ~1.5 seconds
* ZoneAlarm: ~1.0 second

Some tools prioritized fast removal while others performed more comprehensive cleanup processes.

---

## Containment and Lateral Movement

Ransomware samples attempted to spread using network discovery techniques including:

* LLMNR
* NBNS
* mDNS
* SSDP

ARP requests were also observed as the ransomware attempted to identify active hosts within the network.

However, VLAN segmentation prevented the malware from communicating across different network segments.

Endpoint security rules blocking SMB traffic also prevented propagation attempts.

These results demonstrate the effectiveness of network segmentation and layered defenses in containing ransomware outbreaks.
