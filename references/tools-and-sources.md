# Tools and Sources

The following tools and resources were used during the experiment.

## Security Tools

* Wazuh (SIEM monitoring)
* Wireshark (network traffic analysis)
* Procmon (process monitoring)
* Suricata (IDS/IPS)
* pfSense firewall

---

## Attack Simulation Tools

* Hydra (RDP brute force testing)
* Responder (network poisoning and credential capture)

---

## Malware Sources

Ransomware samples were obtained from:

* MalwareBazaar (abuse.ch)
* ANY.RUN malware analysis platform

These repositories provide real-world malware samples used for research and defensive testing.

---

## Monitoring and Analysis

System and network activity were monitored using:

* Wazuh log monitoring
* Windows Event Viewer
* Wireshark packet capture
* Procmon process analysis
